# UI Kit — Website

High-fidelity recreation of the Praktijk van Marijk website, gebaseerd op de aangeleverde brandingpagina. Klikbare prototype met zes pagina's: Home, Over Marijke, Voor professionals, Werkwijze, Ervaringen, Contact.

## Structuur

- `index.html` — entry, laadt React + Babel + alle componenten
- `App.jsx` — router (client-side, simpele state)
- `Topbar.jsx`, `Footer.jsx` — shared chrome
- `Button.jsx`, `Card.jsx`, `Eyebrow.jsx`, `SectionDivider.jsx` — primitives
- `Home.jsx` — hero + herkenning-kaarten + quote-strip + aanbod-grid
- `Over.jsx` — over Marijke, biografie, SKJ-registratie
- `Professionals.jsx` — voor welke rollen, per doelgroep
- `Werkwijze.jsx` — drie-fasen-werkwijze met genummerde kaarten
- `Ervaringen.jsx` — testimonials + quote-strips
- `Contact.jsx` — formulier + NAW
